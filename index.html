Introduction to the Java Multi-Platform and Android SDK

The Java Hue SDK is a set of tools that are designed to make it easy to access the Hue system through the Hue Wi-Fi network connected bridge and control an associated set of connected lamps. The aim of the SDK is to enable you to create your own applications for the Hue system. The tools are provided with documentation for the SDK and example code.

To develop for Hue you require the Philips Hue Personal Wireless Lighting System.

Platform Independent

The Java SDK is platform independent so it will run on any Java Desktop/Android environment. It has been successfully tested on Windows, Linux, OS X and Android (including Libgdx and OUYA, an Android gaming console).

Example Source Code

There are currently 3 sample apps with source code, which demonstrate how to use the SDK:

  • Quick Start App A bare bones Android app with minimal code for connecting to a bridge. A good starting point for creating your own hue app.
  • Sample App A larger example Android app showing more functionality, for example, lights, groups, scenes and schedules.
  • Desktop App A Java Desktop/Application written in Swing with minimal code for connecting to a bridge and changing your lights.
